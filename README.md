# 📝 Full-Stack To-Do List Application with .NET & React

מערכת Full-Stack מלאה לניהול משימות המאפשרת למשתמשים להירשם, להתחבר ולנהל את המשימות האישיות שלהם בצורה מאובטחת. הפרויקט מדגים ארכיטקטורת שרת-לקוח מודרנית, עבודה עם בסיס נתונים וקונטיינרים (Docker).

---

## 🛠️ טכנולוגיות וכלים (Tech Stack)

### **Backend (צד שרת)**
* **שפה וסביבה:** .NET Core (C#) Web API
* **בסיס נתונים ו-ORM:** Entity Framework Core
* **תיעוד ה-API:** Swagger UI
* **DevOps:** Docker & Dockerfile

### **Frontend (צד לקוח)**
* **שפה וסביבה:** React.js (JavaScript) / Node.js
* **ניהול משתני סביבה:** dotenv (.env)

---

## ✨ תכונות מרכזיות (Features)

* **מערכת משתמשים (Authentication):** רישום והתחברות משתמשים (ניהול ישויות User ו-Item בנפרד לכל משתמש).
* **ניהול משימות (CRUD Operations):** יצירה, קריאה, עדכון ומחיקה של משימות.
* **תיעוד API אינטראקטיבי:** שימוש ב-Swagger לבדיקה קלה של ה-Endpoints של השרת.
* **Containerization:** הפרויקט מוכן להרצה בסביבת Docker מבודדת.

---

## 🚀 איך להריץ את הפרויקט מקומית?

### דרישות קדם (Prerequisites)
* SDK של .NET
* Node.js מותקן
* Docker (אופציונלי)

### הרצת השרת (Backend)
1. ניווט לתיקיית ה-Backend.
2. הרצת הפקודה: `dotnet run` או הרצה דרך Docker.

### הרצת הלקוח (Frontend)
1. ניווט לתיקיית ה-Frontend.
2. התקנת תלויות: `npm install`
3. הרצת הפרויקט: `npm start` (או `npm run dev`)
